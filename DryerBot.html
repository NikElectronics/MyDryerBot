Инструкция по управлению myDryerBot

Введение

myDryerBot - это интеллектуальная сушилка для филамента с PID-регулированием температуры и поддержанием влажности.

Подключение устройства

  1. Подключите устройство к источнику питания
  2. Дождитесь инициализации (синий светодиод загорится постоянно)
  3. Подключитесь к Wi-Fi сети "myDryerBot" (пароль: nik170978)
  4. Откройте браузер и перейдите по адресу, который отобразится в приложении
  5. Введите пароль для доступа: nik1709
Примечание: Для постоянной работы рекомендуется подключить устройство к вашей домашней Wi-Fi сети через раздел настроек.

Основной интерфейс

Главный экран содержит следующие элементы управления:

  • Текущие показания температуры и влажности
  • Кнопки управления процессом (Пуск/Пауза/Стоп)
  • Выбор программы и режима работы
  • Индикация состояния

Программы сушки

Устройство содержит предустановленные программы для различных типов филамента и возможность создания собственных программ.

Предустановленные программы

Доступны следующие программы:

  • PLA: 50°C, 15% влажности, 3 часа
  • PETG: 70°C, 15% влажности, 3 часа
  • PA: 80°C, 15% влажности, 3 часа
  • ABS: 85°C, 15% влажности, 3 часа
  • TPU: 60°C, 15% влажности, 3 часа
  • И другие материалы (PVA, PP, ASA)

Собственная программа

Для создания индивидуальной программы:

  1. Перейдите в раздел "Моя программа"
  2. Установите желаемую температуру (0-120°C)
  3. Задайте уровень влажности (0-50%)
  4. Установите время работы
  5. Сохраните настройки

Режимы работы

Автоматический режим

Полный цикл: сушка + поддержание влажности.

  1. Выберите "Авто" в разделе "Выбор режима"
  2. Выберите материал из списка
  3. Нажмите "Пуск"

Режим сушки

Только процесс сушки без поддержания влажности.

  1. Выберите "Сушка" в режимах
  2. Выберите материал
  3. Нажмите "Пуск"

Режим поддержания

Только контроль влажности без предварительной сушки.

  1. Выберите "Поддержание" в режимах
  2. Выберите материал
  3. Нажмите "Пуск"

Ручной режим

Полный контроль всех параметров.

  1. Выберите "Ручной" в режимах
  2. Установите температуру ползунком
  3. Настройте скорость вентилятора
  4. Нажмите "Пуск"

Дополнительные настройки

Настройка PID-регулятора

Для точной настройки температурного контроля:

  1. Перейдите в "Настройка PID"
  2. Включите автонастройку или задайте коэффициенты вручную
  3. P (пропорциональный) - реакция на отклонение
  4. I (интегральный) - устранение статической ошибки
  5. D (дифференциальный) - предотвращение колебаний

Настройка датчиков

Управление системой безопасности:

  • Датчик перегрева - аварийное отключение при превышении температуры
  • Датчик открытия - остановка при открытии крышки

Таймер работы

Автоматическое включение/выключение по расписанию:

  1. Включите функцию таймера
  2. Установите время включения
  3. Задайте время выключения
  4. Сохраните настройки
Важно: Для работы по таймеру устройство должно быть подключено к питанию.

Управление процессом

Запуск процесса

  1. Выберите программу и режим
  2. Нажмите "Пуск"
  3. Зеленый светодиод начнет мигать
  4. Следите за показаниями на экране

Пауза и остановка

  • Пауза - временная остановка с возможностью продолжения
  • Стоп - полное прекращение процесса
Примечание: После полной остановки процесс нельзя возобновить - требуется новый запуск.

Светодиодная индикация

  • Красный мигает - ошибка (перегрев/открытие/датчик)
  • Синий мигает - режим паузы
  • Зеленый мигает - активный процесс
  • Синий горит - готовность к работе

Устранение неисправностей

Ошибки и решения

Ошибка Причина Решение
Перегрев нагревателя Температура превысила безопасный уровень Проверьте вентиляцию, датчик температуры
Крышка открыта Обнаружено открытие во время работы Закройте крышку и перезапустите
Ошибка измерения Проблема с датчиком температуры/влажности Проверьте подключение датчика

Перезагрузка устройства

Если устройство работает нестабильно:

  1. Нажмите "Restart" в разделе WiFi настроек
  2. Или отключите питание на 10 секунд
Важно: При перезагрузке текущий процесс будет прерван.
