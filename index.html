Инструкция по управлению myDryerBot

Полная инструкция для прибора

Ищете подробное руководство по эксплуатации устройства myDryerBot? Ознакомьтесь с полной инструкцией, содержащей информацию о подключении, технических характеристиках и обслуживании прибора.

Открыть инструкцию для прибора

Введение

myDryerBot - это набор для построения собственной интеллектуальной сушилки для филамента с PID-регулированием температуры и поддержанием влажности и возможностью масштабирования на нужное количество катушек филамента (в зависимости от мощности нагревателя).

Подключение устройства

  1. Подключите устройство к источнику питания
  2. Дождитесь инициализации (синий светодиод загорится постоянно)
  3. Подключитесь к Wi-Fi сети "myDryerBot" (пароль: drybot12345)
  4. Если автоматически не запустился WEB интерфейс наберите в браузере 192.168.4.1
  5. При повторном подключении в режиме точка-точка посмотрите IP вашего устройства в настройках.
  6. Введите IP в браузере телефона, планшета или компьютера подключенного к вашей локальной сети.
  7. В режиме точка-точка так же возможна работа.

Основной интерфейс

Главный экран содержит следующие элементы управления:

  • Текущие показания температуры и влажности
  • Кнопки управления процессом (Пуск/Пауза/Стоп)
  • Выбор программы и режима работы
  • Индикация состояния

Программы сушки

Устройство содержит предустановленные программы для различных типов филамента и возможность создания собственных программ.

Предустановленные программы

Доступны следующие программы:

  • PLA: 50°C, 3 часа
  • PETG: 70°C, 3 часа
  • PA: 80°C, 3 часа
  • ABS: 85°C, 3 часа
  • TPU: 60°C, 3 часа
  • PVA: 45°C, 3 часа
  • PP: 70°C, 3 часа
  • ASA: 85°C, 3 часа
  • myProgram: собственна программа

Собственная программа

Для создания индивидуальной программы:

  1. Перейдите в настройки в раздел "Моя программа"
  2. Установите желаемую температуру (0-120°C)
  3. Задайте уровень влажности (0-50%)
  4. Установите время работы
  5. Настройки сохранятся автоматически
  6. В выборе программы выберите myProgram

Режимы работы

Автоматический режим

Полный цикл: сушка + поддержание влажности.

  1. Выберите "Авто" в разделе "Выбор режима"
  2. Выберите материал из списка
  3. Нажмите "Пуск"

Режим сушки

Только процесс сушки без поддержания влажности.

  1. Выберите "Сушка" в режимах
  2. Выберите материал
  3. Нажмите "Пуск"

Режим поддержания

Только контроль влажности без предварительной сушки.

  1. Выберите "Поддержание" в режимах
  2. Выберите материал
  3. Нажмите "Пуск"

Ручной режим

Полный контроль всех параметров.

  1. Выберите "Ручной" в режимах
  2. Установите температуру ползунком
  3. Настройте скорость вентилятора
  4. Нажмите "Пуск"

Дополнительные настройки

Настройка PID-регулятора

Для точной настройки температурного контроля:

  1. Перейдите в "Настройка PID"
  2. Настройте скорость вращения вентилятора
  3. Включите автонастройку или задайте коэффициенты вручную
  4. P (пропорциональный) - реакция на отклонение
  5. I (интегральный) - устранение статической ошибки
  6. D (дифференциальный) - предотвращение колебаний
  7. Наблюдайте процесс на графике.
  8. Добейтесь стабильного поддержания температуры.

Настройка датчиков

Управление системой безопасности:

  • Датчик перегрева - аварийное отключение при превышении температуры
  • Датчик открытия - остановка при открытии крышки

Таймер работы

Автоматическое включение/выключение по расписанию:

  1. Перейдите в настройки
  2. Включите функцию таймера
  3. Установите время включения
  4. Задайте время выключения
  5. Настройки сохранятся авттоматически
Важно: Для работы по таймеру устройство должно быть подключено к питанию.

Управление процессом

Запуск процесса

  1. Выберите программу и режим
  2. Нажмите "Пуск"
  3. Зеленый светодиод на приборе начнет мигать

Пауза и остановка

  • Пауза - временная остановка с возможностью продолжения
  • Стоп - полное прекращение процесса
Примечание: После полной остановки процесс нельзя возобновить - требуется новый запуск.

Светодиодная индикация на приборе

  • Красный мигает - ошибка (перегрев/открытие/датчик)
  • Синий мигает - режим паузы
  • Зеленый мигает - активный процесс
  • Синий горит - готовность к работе

Устранение неисправностей

Сообщения о ошибках появляются на экране (телефона/компьютера).

Ошибки и решения

Ошибка Причина Решение
Перегрев нагревателя Температура превысила безопасный уровень Проверьте вентилятор, датчик температуры
Крышка открыта Обнаружено открытие во время работы Закройте крышку и перезапустите
Ошибка измерения Проблема с датчиком температуры/влажности Проверьте подключение датчика

Перезагрузка устройства

Если устройство нужно перезагрузить:

  1. Нажмите "Restart" в разделе WiFi настроек
  2. Или отключите питание на несколько секунд
Важно: При перезагрузке текущий процесс будет прерван.
